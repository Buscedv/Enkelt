ÖVERSIKT

Om () och Annars går hand i hand och fungerar tillsammans. Om () låter dig testa olika saker. Du kan t.ex. kolla om en variabel är lika med en annan variabel, om ett nummer är större än ett annat nummer osv. Om uttrycket i Om () stämmer körs koden inuti Om () medan koden inuti Annars bara körs om uttrycket i Om () är falsk/inte stämmer. Du kan bara använda Annars om du också använder Om (), men du kan använda Om () ensamt.

Syntax

Om

om (data = data) {

        # kod här

}

om : Omfunktionen det som är innanför pareneserna “(“ “)” är uttrycket.

data : Kan vara ett nummer, en textsträng, variabel etc…

= : Uttryck, kan vara: =, !, < eller >.

data : Ett annat nummer, annan textsträng, variabel, etc…

} : Om startparentes, markerar starten på Om () “parentesblocket”.

# : En kommentar som representerar kod inuti Om () “parentesblocket”.

} : Om slutparentes, markerar slutet på Om () “parentesblocket”.

Annars

om (data = data) {

        # kod här

}

annars {

        # kod här

}

om, data, =, data, {, }, # : Om () syntax.

annars : annars funktion.

{ : Annars startparentes, markerar starten på Annars “parentesblocket”.

#: En kommenentar som representerar kod inuti Annars “parentesblocket”

} : Annars slutparentes, markerar slutet på Annars “parentesblocket”.

Uttryck

Du kan använda dig av några olika uttryck i Om ().

Uttryck

Beskrivning

=

Lika med/samma

!

Inte lika med/olika

<

Mindre än (bara nummer)

>

Större än (bara nummer)

Booleskvariabler

Du behöver inte skriva ett uttryck om du använder en booleskvariabel.

Exempel

Här är ett exempel på hur Om () & Annars kan användas i din kod.

Det här programmet frågar användaren efter hens namn. Om namnet är “Jonas” skriver programmet “Hej Jonas” och annars (om namnet inte är “Jonas”) skriver programmet “Hej” tillsammans med det namn som användaren skrev in.

Kod

1 var namn = in(“Skriv ditt namn: “)

2 om ($namn = “Jonas”) {

3 skriv (“Hej Jonas!”)

}

4 annars {

        5 var svar = “Hej “ + $namn

        6 skriv ($svar)

7 }

Utput ex1 (Användaren skrev Jonas)

Skriv ditt namn: Jonas

Hej Jonas!

Utput ex2 (Användaren skrev Edvard, alltså inte Jonas)

Skriv ditt namn: Edvard

Hej Edvard

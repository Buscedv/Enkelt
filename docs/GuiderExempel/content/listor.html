ÖVERSIKT

Listor är en variabeldatatyp där du kan spara flera texter, nummer etc. i samma variabel och under samma variabelnamn.

Ex.

Listan inköp ser ut så här: ["bröd", "ägg", "mjölk"] Om vi kallar listan inköp, kan man skriva ut hela listan med att skriva: skriv ($inköp) Då ser man hela listan, men om du vill få ut en skild sak från listan måste du specificera sakens indexnummer. Varje sak i listan har ett eget indexnummer. 'bröd' har då indexen 0 eftersom 'bröd' är den första saken i listan och listor börjar från 0 inte 1!

Syntax

  • Listor är en variabeldatatyp.
  • Listor deklareras med [] parenteser.
  • Allting inne i en lista kallas för saker.
  • Saker uppdelas med kommatecken (,).
  • Varje sak har ett eget indexnummer.
  • Den första saken i en lista har indexnumret 0.

Skapa ny lista

Du skapar en ny lista på samma sätt som du skapar en variabel:

var min_lista = [“sak1”, “sak2”, “sak3”, 1, 2, 3, “sak4”]

var : nyckelordet för att deklarera variabler.

min_lista : listans/variabelns namn.

[ : Visar var listan börjar.

“sak1” - “sak3” : saker i listan med datatypen text.

1-3 : saker i listan med datatypen nummer.

“sak4” : sak i listan.

] : Visar var listan slutar.

Saker i listor kan vara olika datatyper:

  • Text/Strings
  • Nummer
  • Boolesk (Sant/Falskt)

Arbeta Med Listor

Visa lista

Du kan skriva/visa hela listor med skriv () funktionen:

skriv ($min_lista)

Utput

[“sak1”, “sak2”, “sak3”, 1, 2, 3, “sak4”]

Visa sak från lista

Du kan skriva/visa en specifik sak från en lista med att specificera ett indexnummer.

skriv ($min_lista[2])

Utput

sak3

Lägg till sak i lista

Du kan lägga till saker i en lista med hjälp av till () funktionen. Du använder den så här:

Ex.

var lista = [“hej”, “världen”]

skriv ($lista)

$lista.till(“!”)

skriv ($lista)

Utput

[‘hej’, ‘världen’]

[‘hej’, ‘världen’, ‘!’]

Ta bort sak från lista

Du kan ta bort saker från en lista med bort () funktionen. Du använder den så här:

Bort () funktionen behöver ett indexnummer som den ska ta bort.

Ex.

var lista = [“hej”, “världen”]

skriv ($lista)

$lista.bort[1]

skriv ($lista)

Utput

[‘hej’, ‘världen’]

[‘hej’]

Längd på lista

Du kan använda längd () funktionen för att ta reda på hur lång en lista är. Längd () funktionen berättar hur lång en rad med tecken är om du använder den på t.ex en textsträng eller ett nummer men om du använder funktionen på en lista returnerar den hur många saker det finns i listan.

Ex.

var lista = [“hej”, “världen”]

var x = längd($lista)

skriv ($x)

Utput

2

Exempel

Här är ett exempelprogram som visar hur du kan använda listor i dina program.

Programmet ber om två indexnummer från användaren. Programmet tar sedan sakerna med de indexnumren användare gav och dividerar dem med varandra.

Kod

1 var lista = [1, 6, 3, 5]

2 var index = in(“Välj index för det första numret 0-3: ”)

3 var num1 = $lista[$index]

4 var index = in(“Välj index för det andra numret 0-3: ”)

5 var num2 = $lista[$index]

6 matte ($num1/$num2)

Utput

Välj index för det första numret 0-3: 2

Välj index för det andra numret 0-3: 0
